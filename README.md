# FoodUtils

A quality-of-life RuneLite external plugin that displays healing values directly on food items in your inventory.

## Features

- Displays food healing amounts on inventory items
- Shows:
  - Base heal
  - Effective heal
  - Both (configurable)
- Color indicators for:
  - Full value heal
  - Wasted heal
  - No heal
- Supports:
  - Standard food
  - Anglerfish (overheal logic)
  - Saradomin brews
